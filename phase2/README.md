# DeelTech Solutions — Phase 2 Delivery

This package contains the Faculty Account Automation System ready for deployment.

- **`scripts/main.sh`** — The application script. Make executable with `chmod +x scripts/main.sh` and run with `sudo ./scripts/main.sh`.
- **`docs/user_manual.md`** — Complete user manual covering installation, license activation, all menu options, troubleshooting, and security guidance.

Refer to the user manual for detailed instructions on system requirements, first-run setup, and usage.
